# 2048Solver
2048 webapp and solver (bot) in python

### Uses expectimax 
Similar to minimax but instead of minimising, calculate sum (probability * evaluation score)

Default depth: 2 if freespaces in grid < 5 else 3

Score: averages >2048

### Dependencies:
 - pynput
 - flask
 - numpy

### How to use:
1. Run `web.py`
